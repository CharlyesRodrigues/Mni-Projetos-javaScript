# 01 Lâmpada

Neste  vídeo vemos como criar uma lâmpada com três ações:

+ LIGAR 
- DESLIGAR 
+ QUEBRAR


Para isso temos que interagir com elementos HTML, eventos e utilização de funções, sempre tentando criar um código semântico.
